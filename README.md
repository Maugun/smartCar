# SmartCar

**SmartCar** is a test project for self driving cars. Cars learn how to drive using an **Unsupervised Neural Network (UNN)** & a **Generic Algorithm**.

### Prerequisites

* Install Unity3D

## Built With

* [Unity3D](https://unity3d.com/) - Game Development Platform
